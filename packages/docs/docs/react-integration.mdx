# Mounting to the DOM

VizCraft renders through the builder's \`mount\` method. You create a builder, point it at a DOM container, and VizCraft handles the SVG render.

## Basic Mount

<CodePreview code={`
import { viz } from 'vizcraft';

const builder = viz()
  .view(400, 200)
  .node('left').at(80, 100).circle(18).label('Left')
  .node('right').at(320, 100).rect(60, 40, 8).label('Right')
  .edge('left', 'right')
    .arrow()
    .done();

const container = document.getElementById('viz-mount');
if (container) builder.mount(container);
`}>
  <VizMount
    builder={mountBuilder}
    style={{ height: '200px', width: '100%', border: '1px solid #eee' }}
  />
</CodePreview>
