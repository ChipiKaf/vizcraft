# Interactive Examples

Here you can see **VizCraft** in action.

## Basic Scene

This example demonstrates a simple scene with a single node.

<CodePreview code={`
import { viz } from 'vizcraft';

const style = document.createElement('style');
style.textContent = \`
  .center-text text {
     text-anchor: middle;
     dominant-baseline: middle;
  }
  .outline-node .viz-node-shape {
     fill: transparent;
     stroke: black;
     stroke-width: 2px;
  }
\`;
document.head.appendChild(style);

const builder = viz()
  .view(800, 600)
  .node('node1')
    .at(400, 300)
    .circle(100)
    .label('Hello VizCraft', { dx: 0, dy: 5, className: 'center-label' })
    .class('center-text outline-node')
    .done();

const container = document.getElementById('viz-basic');
if (container) builder.mount(container);
`}>
  <div style={{ height: '400px', width: '100%', position: 'relative' }}>
    <style>{`
      .center-text text {
         text-anchor: middle;
         dominant-baseline: middle;
      }
      .outline-node .viz-node-shape {
         fill: transparent;
         stroke: black;
         stroke-width: 2px;
      }
    `}</style>
    <VizMount builder={basicBuilder} style={{ height: '100%', width: '100%' }} />
  </div>
</CodePreview>
